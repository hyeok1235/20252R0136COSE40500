# Amazon Web Service: Living in the AI/ML Era

**Speaker:** 김대근 (Dae-Keun Kim)

**GitHub:** github.com/daekeun-ml

## Industry Trends

- **Shift from ML to Agentic AI:** Current focus is on agentic approaches rather than traditional ML
- **Infrastructure role:** Efficiently distributing resources received from NVIDIA is key

## Career Documentation: STARF Format

Structure your outcomes on one page using:

| Component | Description |
|-----------|-------------|
| **S**ituation | Context and background of the challenge |
| **T**ask | Specific responsibility or goal |
| **A**ction | Steps taken to address the task |
| **R**esult | Quantifiable outcomes achieved |
| **F**eedback | Lessons learned and improvements |

## Key Takeaways

- Agentic AI is the current industry focus over traditional ML
- AWS's role centers on resource distribution and deployment
- STARF format helps structure and communicate professional achievements effectively
